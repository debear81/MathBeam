# MathBeam worksheet for MathCAD
MathBeam is a MathCAD Prime (10) worksheet for one-dimensional beam analysis using the stiffness method.

## FEATURES
- Single & multi-span beam analysis
- Point loads
- Reaction, shear, and moment output

## CAPBILITIES & ASSUMPTIONS
- Single & multi-span beam analysis
- Linear elastic analysis
- Small displacement theory
- Pin supports assumed

## LIMITATIONS
This is a work in progress, beta version. Current version limitations include:
- No fixed, roller, or spring support
- No applied moment
- No applied distributed loads
- No Timoshenko (shear deformation) analysis
- No dynamic analysis
- No (automatic) moving load analysis
- No influence line generation
- No geometric nonlinearity
- No material nonlinearity
- No frame or truss analysis
